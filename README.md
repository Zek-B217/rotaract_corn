## 1. Installazione e Avvio

Queste istruzioni ti permetteranno di ottenere una copia funzionante del progetto in locale per scopi di sviluppo e testing.

### Prerequisiti

* XAMPP
* VS Code

### Istruzioni di Installazione

1.  **Attivare Apache da XAMPP**

2.  **Clona il repository nella cartella htdocs di XAMPP:**
    ```bash
    git clone https://github.com/DaniPopCorn/rotaract_corn.git
    cd rotaract_corn
    ```

4.  **Visita il sito in locale usando localhost**
    

## 2. Stack Tecnologico

Il progetto è costruito utilizzando le seguenti tecnologie principali:

* **Linguaggio Backend: PHP** 
* **Linguaggi Frontend: HTML5, CSS, JavaScript**

## 3. Struttura delle Directory

Una panoramica delle directory principali del progetto:
* **/**: contiene la pagina index.html, le pagine admin, la LICENSE di GitHub e questo README.md
* **/CSS**: contiene una serie di file CSS, uno generale comune ad ogni pagina (commonStyle.css) e uno specifico dedicato alle specifiche pagine (homeStyle.css, whoWeAreStyle.css, …).
* **/Documenti**: contiene i vari documenti riguardanti il sito e la sua realizzazione come l’analisi dei requisiti, WBS, Gantt, relazione tecnica e presentazione.
* **/JS**: contiene i file JS che sono serviti per dare della dinamicità al nostro sito.
* **/Media**: contiene le foto presenti nel sito.
* **/PHP**: contiene le costanti per il PHP.
* **/PHPMailer**: Progetto clonato da https://github.com/PHPMailer/PHPMailer per poter inviare la mail dalla pagina contatti.php
* **/Pages**: contiene tutte le pagine del sito
